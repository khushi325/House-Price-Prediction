# 🏠 House Price Prediction System

## 📌 Project Overview

This project predicts house prices using Machine Learning techniques. The model is trained on the House Price India dataset containing 14,620 records and 23 features. After preprocessing and feature selection, a Random Forest Regressor was selected as the final model due to its high prediction accuracy.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization using Matplotlib and Seaborn
- Multiple Machine Learning Models
- Model Comparison
- Random Forest Regressor
- House Price Prediction
- Interactive Streamlit Web Application

---

## 📊 Dataset Information

- Dataset: House Price India
- Total Records: 14,620
- Features Used: 19
- Target Variable: Price

---

## 🤖 Machine Learning Models Used

- Linear Regression
- Decision Tree Regressor
- Gradient Boosting Regressor
- Random Forest Regressor (Final Model)

---

## 📈 Model Performance

| Model | R² Score |
|--------|---------:|
| Linear Regression | 0.7186 |
| Decision Tree | 0.7455 |
| Gradient Boosting | 0.8648 |
| Random Forest | **0.8759** |
